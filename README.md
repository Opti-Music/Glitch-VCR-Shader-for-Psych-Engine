# Glitch-VCR-Shader-for-Psych-Engine
Gamebreaker type shader (fnf reference)

To use the shader, create a new file named vcrGlitch.frag and place it inside your mod's folders at mods/shaders/.
Create a file named glitchController.lua and save it inside your specific song's folder (e.g., mods/songs/your-song/). This script manages the inputs and provides custom events/functions to safely turn it on and off.
